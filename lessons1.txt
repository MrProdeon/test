Для отображения информации о папке в GitBash используется команда ls

Для отображения информации о папке в терминале используется команда dir

Чтобы перейти в папку, необходимо написать команду cd и название папки
Чтобы перейти в предыдущую папку, необходимо написать команду cd ..

Чтобы превратить папку в репозиторий Git - необходимо прописать команду git init , либо в среде разработки в разделе ГИТ
